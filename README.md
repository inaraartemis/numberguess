# 🌳 Binary Search Tree Visualizer (Dtreenumber)

A simple, interactive Binary Search Tree (BST) visualizer built with Python, Gradio, and Matplotlib. This application allows users to play a "Guess My Number" game while visualizing how a binary search tree is constructed step-by-step.

## 🚀 Features
- **Interactive UI**: Built with Gradio for a seamless web-based experience.
- **Dynamic Visualization**: Real-time tree rendering using Matplotlib.
- **Step-by-Step Construction**: Watch the BST grow as you answer questions.
- **Game Logic**: Uses binary search principles to guess your number between 0 and 10.

## 🛠️ Technology Stack
- **Python**: Core logic.
- **Gradio**: Frontend interface.
- **Matplotlib**: Tree visualization and plotting.

## 🏃 How to Run
1.  **Install Dependencies**:
    ```bash
    pip install gradio matplotlib
    ```
2.  **Run the Application**:
    ```bash
    python main.py
    ```
3.  **Play**: Open the local URL provided by Gradio (usually `http://127.0.0.1:7860`) and start the game!

## 📂 Project Structure
- `main.py`: Contains the game logic, tree data structure, and Gradio interface.
